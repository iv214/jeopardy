# Frontend

## Установка зависимостей

Открыть терминал в данной директории, ввести команду:

> npm install --legacy-peer-deps

## Настройка
1. Создать копию .env_example
2. Переименовать созданный файл в .env
3. Открыть файл с помощью текстового редактора
4. Заполнить отсутствующие поля

## Запуск
Открыть терминал в данной директории, ввести команду запуска:

> npx vite --host

## Обновление зависимостей

> npm update --legacy-peer-deps
